**Toolbar Breakdown:**
```
H1 ▾    = Headers (H1, H2, H3, Normal)
B       = Bold
I       = Italic
U       = Underline
S       = Strikethrough
📝      = Numbered list
•       = Bullet list
≡       = Align left
⎯       = Align center  
≣       = Align right
←       = Decrease indent
→       = Increase indent
A ▾     = Font size (Small, Normal, Large, Huge)
🧹      = Clear formatting
```
